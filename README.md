# kakucho（かくちょ）

ブランド品せどり向け確定申告・帳簿補助ツール。

## できること
- kobuchoのデータを取り込んで棚卸を自動計算
- 売上原価を棚卸法で自動算出
- 収支内訳書をそのまま印刷・CSV出力
- 経費の記録とカテゴリ別集計

## インストール手順
1. zipを解凍してデスクトップに置く
2. コマンドプロンプトで以下を実行（初回のみ）
   cd %USERPROFILE%\Desktop\kakucho
   npm install
3. start.bat をダブルクリックして起動
4. ブラウザで http://localhost:3941 が開きます

## kobuchoとの連携
設定タブ → kobuchoのDBパスを指定 → 取り込みボタン

## 動作環境
- Windows 10/11
- Node.js 18以上
